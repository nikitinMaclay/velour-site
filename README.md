## 🚀 Пошаговый деплой (15 минут)

### Шаг 1: Создай Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Дай имя боту (например: `Velour Orders Bot`)
4. Дай username (например: `velour_orders_bot`)
5. **Скопируй токен** — он выглядит так: `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxx`

### Шаг 2: Узнай свой Chat ID

1. Найди в Telegram бота **@userinfobot**
2. Напиши ему `/start`
3. Он ответит твоим **Chat ID** — это число вроде `123456789`
4. ⚠️ **Важно:** после этого открой своего бота (из Шага 1) и напиши ему `/start` — без этого бот не сможет тебе писать!

### Шаг 3: Залей на GitHub

1. Зайди на [github.com](https://github.com) → создай аккаунт (если нет)
2. Нажми **New Repository** → назови `velour-site` → Create
3. Загрузи все файлы из этой папки (можно через кнопку "Upload files" на GitHub)

### Шаг 4: Деплой на Netlify

1. Зайди на [netlify.com](https://www.netlify.com) → Sign up через GitHub
2. Нажми **"Add new site"** → **"Import an existing project"**
3. Выбери GitHub → найди свой репозиторий `velour-site`
4. Netlify автоматически подхватит настройки из `netlify.toml`
5. Нажми **Deploy** — через минуту сайт будет онлайн!

### Шаг 5: Добавь секреты (токен бота)

1. В Netlify зайди в свой сайт → **Site configuration** → **Environment variables**
2. Добавь две переменные:

   | Key                   | Value                              |
   |-----------------------|------------------------------------|
   | `TELEGRAM_BOT_TOKEN`  | Токен из Шага 1                    |
   | `TELEGRAM_CHAT_ID`    | Chat ID из Шага 2                  |

3. Нажми **Save**
4. Зайди в **Deploys** → **Trigger deploy** → **Deploy site** (чтобы переменные подхватились)

### Шаг 6: Проверь!

1. Открой свой сайт (ссылка вида `https://velour-site.netlify.app`)
2. Заполни форму обратной связи → отправь
3. Проверь Telegram — должно прийти сообщение от бота! 🎉

## ❓ FAQ

**Q: Это бесплатно?**
A: Да! Netlify бесплатен до 125K вызовов функций/месяц. Telegram API тоже бесплатный.

**Q: Могу ли я добавить свой домен?**
A: Да. В Netlify → Domain management → Add custom domain.

**Q: Как добавить/убрать товары?**
A: В `index.html` найди массив `PRODUCTS` в самом начале `<script>` — там можно менять названия, цены и картинки.

**Q: Что дальше? (когда захочется большего)**
- Админ-панель для управления товарами → Supabase / Firebase
- Оплата онлайн → Kaspi Pay / Stripe
- CMS для контента → Strapi / Sanity
